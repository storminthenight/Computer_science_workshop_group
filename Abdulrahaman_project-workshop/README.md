This folder has my Python generated individual data visualizations. There are two main plots using the dataset; A Line Plot which shows the trend in popularity of two games over Time; A Bar Chart showing the Top Six Most Searched Games by Popularity to Compare Overall Popularity. The Visualizations aid in Trend Identification and Game Popularity Comparisons.
These visualizations were created with Python Libraries. Pandas was used for Data Manipulation & Cleaning, Matplotlib was utilized to create the Charts. Data was retrieved from an excel document prior to use. In order to Run the Code, Open the File, Ensure your Dataset is within the Same Directory as the python Script, Run the python Script to Display the Plots.
